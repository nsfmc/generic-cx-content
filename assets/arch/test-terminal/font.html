kricket

   I started working on my very first font on June 28 2004.

   Download the font: kricket.ttf

     The current version (1) has all the numbers, letters (upper and
  lowercase) and some punctuation.  It is not, however, properly kerned.  Wait
  for the next release...

     The font is based on the handwriting of my friend kate.  You can see her
  picture in the showoff dir, it's kate.jpg.
     The name of the font is Kricket.  It is pronounced exactly like the small
  insect "cricket".  The name kricket was first coined by my friend deb.

  You can make your own fonts too.  Check out fontforge.
